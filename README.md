# Claude Code Configuration

This repository contains my personal settings and custom skills for [Claude Code](https://github.com/anthropics/claude-code).

## Contents

- `settings.json` - Claude Code configuration and preferences
- Custom skills and extensions for enhanced functionality

## Usage

To use these configurations:

1. Clone this repository
2. Copy `settings.json` to your Claude Code settings directory
3. Install any custom skills as needed

## About Claude Code

Claude Code is Anthropic's official CLI tool for Claude, providing an interactive command-line interface for software engineering tasks.

Learn more at the [Claude Code repository](https://github.com/anthropics/claude-code).
